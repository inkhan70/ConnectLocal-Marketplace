
# ConnectLocal: Professional B2B & B2C Marketplace

ConnectLocal is a robust, high-performance platform designed to bridge the gap between local producers, wholesalers, distributors, shopkeepers, and customers.

## 🌟 Key Features
- **City-Specific Search:** Advanced filtering to find businesses in your exact vicinity.
- **Proximity-Based Discovery:** Automatic sorting of listings within a 100km radius using geolocation.
- **Secure Transaction Verification:** Unique 14-digit pickup codes for safe product retrieval.
- **Loyalty Rewards:** Integrated "Ghost Coin" system to reward frequent buyers.
- **AI-Powered Branding:** Instant product descriptions and business slogans powered by Genkit AI.
- **Multi-Language Support:** Native support for English, Arabic, Farsi, Urdu, and Hindi.

## 🛠 Tech Stack
- **Framework:** [Next.js 14](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) + [ShadCN UI](https://ui.shadcn.com/)
- **Database & Auth:** [Firebase](https://firebase.google.com/) (Firestore / Storage / Auth)
- **AI Integration:** [Genkit AI](https://firebase.google.com/docs/genkit) (Google Gemini 2.0 Flash)

## 🚀 Development Roadmap
This project was prototyped in Firebase Studio and is ready for production scaling. 

### Recommended Next Steps:
1. **Payment Integration:** Set up **Stripe** or a local mobile money gateway (M-Pesa, etc.) for secure transactions.
2. **Mobile Experience:** Convert the app into a **Progressive Web App (PWA)** for home-screen installation.
3. **Notifications:** Implement **Firebase Cloud Messaging (FCM)** for real-time order and chat alerts.
4. **Escrow Logic:** Enhance the order system to hold funds until the pickup code is verified by the seller.

## 📦 Local Setup
1. **Clone the Repo:** `git clone <your-repo-url>`
2. **Install Dependencies:** `npm install`
3. **Run Dev Server:** `npm run dev`
4. **Build Check:** `npm run build`
